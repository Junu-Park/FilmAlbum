# FlimAlbum

## 영화에 대한 **검색** 및 **상세정보**, **좋아요** 기능을 제공하는 1인 프로젝트 앱 (2025/01/24~2025/02/01)

***2. Cinema 탭: 프로필, 최근 검색어 내역, 오늘의 영화 정보를 확인 가능합니다.***

* 프로필 배너에서는 설장한 닉네임, 이미지, 그리고 좋아요한 영화 개수를 확인할 수 있고 배너를 클릭해서 프로필을 편집할 수 있습니다.
* 상단 돋보기 버튼과 최근 검색어를 클릭해서 검색 화면으로 이동할 수 있습니다.
* 오늘의 영화를 클릭해서 해당 영화의 상세정보 화면으로 이동하거나 하트 버튼을 눌러서 영화에 좋아요를 할 수 있습니다.

***3. Search 화면: 영화를 검색하고 상세정보를 확인할 수 있습니다.***

* 검색 후, 영화를 클릭해서 상세정보 화면으로 이동하거나 하트 버튼을 눌러서 영화에 좋아요를 할 수 있습니다.
* 상세정보 화면에서는 백드롭, 개봉일, 별점, 장르, 시눕시스, 캐스트, 포스터와 같은 정보를 확인할 수 있고 상단의 하트 버튼을 눌러서 영화에 좋아요를 할 수 있습니다.

***5. Profile 탭: 프로필 확인과 탈퇴를 할 수 있습니다.***
